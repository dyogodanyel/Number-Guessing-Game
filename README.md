# Number Guessing Game

Link do desafio: [Number Guessing Game – Roadmap.sh](https://roadmap.sh/projects/number-guessing-game)

## Descrição
Um jogo simples de adivinhação de número via linha de comando. O computador escolhe um número aleatório entre 1 e 100, e o usuário deve adivinhar qual é esse número dentro de um número limitado de tentativas, definido pelo nível de dificuldade.

## Funcionalidades
- Exibe uma mensagem de boas-vindas e as instruções ao iniciar.  
- Permite que o usuário escolha o nível de dificuldade (easy, medium, hard), definindo quantas tentativas ele terá.  
- Gera um número aleatório entre 1 e 100 para o usuário adivinhar.  
- A cada tentativa, informa se o palpite está acima ou abaixo do número alvo.  
- Termina o jogo quando o usuário acerta ou quando as tentativas se esgotam.  
- Ao acertar, exibe mensagem de parabéns e o número de tentativas usadas.
